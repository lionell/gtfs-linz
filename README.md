To start the server run: `python -m http.server` in this directory. (Replace with `python3` if that's what you have).
Navigate to `localhost:8000` in the browser.

If it's too slow try shrinking `stops.txt` and `shapes.txt` files.

Link: [https://lionell.github.io/gtfs-linz](https://lionell.github.io/gtfs-linz)
